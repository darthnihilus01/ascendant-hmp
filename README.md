# Ascendant 2026

Keyboard-first desktop experience for the Ascendant 2026 hackathon.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

## Keybindings

- alt + 1..3 switch workspace
- alt + left/right focus
- alt + space launcher
- alt + s status bar
- alt + m minimize
- esc reset focus

## Structure

- src/app
- src/components/desktop
- src/state
